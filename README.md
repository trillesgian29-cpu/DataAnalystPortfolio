## 📊 Business Understanding
The goal of this portfolio is to showcase my ability to apply the **Data Analysis Lifecycle**:
1. Define questions clearly.  
2. Collect and prepare data.  
3. Analyze using SQL, Python, and visualization tools.  
4. Present results in recruiter-friendly formats.  

I selected projects that highlight both technical skills and clear communication for non-technical audiences. Challenges included learning SQL join logic, handling outliers, and designing dashboards that tell a story.
